Showing your underscore in public can be embarrassing, put on some underwear.

Underscore.js is a great library, but I prefer to extend the native prototypes with these methods since extensive use of underscore makes some code difficult to read. With Underwear.js, you can use the greatness of underscore.js from the comfort of your own objects.

Underwear translates most of the Underscore methods to the native JavaScript prototypes (excluding Objects). For those who prefer not to see skid marks in their code ;)

With Underwear you can do this:

[1, 2, 3].each(function(n) { alert(n); });

Instead of this:

_([1, 2, 3]).each(function(n) { alert(n); });

Or this:

_.each([1, 2, 3], function(n) { alert(n); });
